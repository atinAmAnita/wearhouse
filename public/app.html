Printer setup help
System printer (AirPrint / Mopria):
  • iPhone: the printer must support AirPrint. Make sure it's on the same Wi-Fi. Tap System Print → pick it in the iOS print sheet.
  • Android: the printer must support Mopria (or have its plugin installed). Same Wi-Fi → tap System Print → pick it.
  • The app can't see or auto-pick the printer (the OS handles that) — just choose your label size above so it prints the right dimensions.
Bluetooth label maker (SUPVAN, etc.):
  • Tap Save / Share → choose the SUPVAN app in the share sheet (or “Save to Photos”, then import it in SUPVAN) → print from there.
  • Pair the printer once inside the SUPVAN app itself, not in the browser.
